
# protobuf安装
官方github下载release版本 2.6.1 

安装步骤在这里:
https://github.com/google/protobuf/tree/master/src
当./autogen.sh的时候:
    会提示报错 gtest-1.5.0.tar.bz2 不能成功下载
    可用下载地址 http://www.filewatcher.com/m/gtest-1.5.0.tar.bz2.653338-0.html
    下载之后按照./autogen.sh步骤做修改，并屏蔽下载功能。
    需要sudo ./autogen.sh
剩下的步骤可以按照官方文档操作。
